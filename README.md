# Welcome to Nutq 👋

Nutq is an open-source speech therapy platform built to empower speech-language pathologists and patients through digital rehabilitation, clinical analytics, and AI-ready technologies.

## Our Mission

To improve speech therapy outcomes by making rehabilitation measurable, accessible, and data-driven.

## Projects

- Nutq Platform
- Documentation
- Future AI Models

## Tech Stack

- ASP.NET Core
- React
- PostgreSQL
- TypeScript
- Tailwind CSS

We're always open to collaboration from developers, researchers, and healthcare professionals.
